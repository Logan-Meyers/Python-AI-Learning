# Python-AI-Learning
Some experiments with creating ML models, LLMs, and possibly chatbots in Python. Based on various courses.

### Setup

1. Make sure [Python](https://www.python.org/) is installed on your computer; version **3.11+ required**.
2. Navigate to the roof of this repository in the terminal of your choice.
3. run `pip install -r requirements.txt` to install all required libraries for all projects in here.
4. Navigate to the desired project directory.
5. Run `python main.py` to run the project you'd like to run.
